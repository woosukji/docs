브라우저 상에서 사전에 정의한 컴포넌트를 클릭할 경우 자동으로 인식합니다. 어떤 조건의 컴포넌트를 클릭한 것인지 의도를 더욱 자세하게 설정할 수 있습니다.

## 1. 컴포넌트 액션 상세조건 설정하기

액션 왼쪽의 **\[>\]** 버튼을 눌러 컴포넌트 상세조건을 설정합니다.

컴포넌트 상세조건은 아래와 같은 형식입니다.

```bash
<컴포넌트> where <컴포넌트 속성> <비교 연산자> <비교 대상 값>
```

<Note>
  **컴포넌트 상세조건이 없을 때, 조건 추가** <br />
  해당 줄에 마우스를 올리면 나타나는 **\[+\]** 버튼을 통해 추가할 수 있습니다.
</Note>
