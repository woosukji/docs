## \[프로젝트 화면 상세 기능 안내\]

<img className="block" src="/assets/create/tests-page.png" />

1. 내비게이션 바
2. 테스트 케이스를 생성하고 실행할 수 있습니다.
3. 만들어진 테스트 케이스의 리스트입니다.

## 1. 테스트 케이스 생성하기

우상단의 **\[Create Test\]** 버튼을 누르고, 테스트 케이스 이름과 시작 URL을 입력해 테스트케이스를 생성할 수 있습니다.

<Note>
자주 사용되는 URL이라면, 텍스트를 드래그한 후 **\[Set as Variable\]** 버튼을 눌러 [환경변수로 저장](/guide/write/env-variable)하세요. <br />
변수 저장 이후, 입력창에 `{`(중괄호) 를 입력해서 다시 사용할 수 있습니다.
</Note>
