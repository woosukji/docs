Reliv 에디터 **\[Test Steps\]** 패널 우상단의 **\[+\]** 버튼을 눌러 특수 액션을 추가할 수 있습니다.

특수 액션의 종류는 다음과 같습니다.

- Check 액션 : 현재 상태가 기대한 상태와 같은지 체크하는 Assertion 입니다.
  - `isVisible` : 특정 요소가 화면에 표시되는지
  - `isEqualTo` : 특정 요소의 현재 텍스트가 정확한지
  - `Component` : 컴포넌트가 화면에 표시되는지
- Add 액션
  - `Custom Action` : Playwright 코드로 액션을 정의할 수 있습니다.
  - `Action Block` : 자주 사용하는 스텝들을 묶은 액션블록을 추가합니다.
  - `Browser Action` : URL 이동, 새로고침과 같은 브라우저 액션입니다.
- `Create Component` : 화면 내에 반복적으로 나타나는 요소를 컴포넌트로 등록합니다.
